ECHOACCESS

Cross-platform configuration file synchronization with field-level encryption, 3-way merge, and approval workflows.

[ PROFILE SYNC ]

Per-device TOML profiles with field-level overrides, transforms, and masking rules. Sync SSH configs, dotfiles, and app settings.

[ DUAL ENCRYPTION ]

age whole-file encryption for sensitive keys. AES-256-GCM field-level encryption for structured configs. argon2 key derivation.

[ 3-WAY MERGE ]

Automatic conflict detection with approval queues. No changes pushed without your consent. Full merge history preserved.

[ TRIPLE INTERFACE ]

CLI for scripts and automation. NieR-styled TUI for interactive management. REST API for integration and remote access.

[ CLOUD STORAGE ]

Pluggable CloudBackend trait. S3-compatible out of the box — works with Aliyun OSS, AWS S3, MinIO, Cloudflare R2.

[ SSH DEVICE PUSH ]

Discover devices from ~/.ssh/config. Push encrypted configs to remote hosts. Bootstrap new devices in one command.
